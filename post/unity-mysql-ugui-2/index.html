使用UGUI与MySql实现的简单角色管理系统

2017-06-05

记录一下大三的数据库项目,虽然挺垃圾的🙃


实现工具：unity5.5 、 MySQL 、Visual Studio 2015 、 Wampserver 3.0.6
实现结果：游戏界面客户端，游戏数据管理查询客户端。

游戏界面客户端：

功能：登录注册、角色选择与创建、已有角色选择、商城购买装备、添加删除好友、背包装备与金币显示。

功能：显示所选角色种类以及背包

商城界面：
功能：点击装备显示价格、右键点击购买并添加至背包。

好友界面：
